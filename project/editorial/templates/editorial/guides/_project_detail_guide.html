Clicking the plus sign will provide an interfacet to create a new entry in that section.

(All )The project page can only show a few of the most recent entries in a given section. Clicking this link will take you a page with all of the entries for a given section that are associated with this project.

Clicking the pencil icon will take you to a page where you can edit the details for this project, including its description, status, team and partners.

Clicking the trash icon will delete a project. You will get a confirmation screen, to ensure you want to delete the project. Deleting a project only deletes the project itself, not the stories or assets associated with it. Deleted projects cannot be recovered.
